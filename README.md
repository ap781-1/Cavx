Welcome to the repository for our novel CaviaR Model with HFD implementation. This function is based on the CAViaR model developed by Engle and Manganelli (2004).

This repository contains code that extends the original Conditional Autoregressive Value at Risk (CAViaR) model by incorporating High-Frequency Data (HFD) to enhance financial risk estimation. Following the approach of Engle and Manganelli (2004), this function implements a CAViaR model for estimating Value at Risk.

Features:
Implementation of multiple CaviaR and GARCH model specifications with HFD components.
Utilization of high-frequency financial data to improve risk prediction accuracy.
Optimization techniques for parameter estimation.
Performance evaluation with backtesting methodologies.
